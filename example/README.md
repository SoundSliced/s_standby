# s_standby Example

A minimal Flutter app that demonstrates `MyAwaitOverlay` from the `s_standby` package.

## Run

From the package root:

```bash
cd example
flutter pub get
flutter run
```

## What to expect

- **Run success demo**: shows a progress overlay for ~2 seconds, then closes.
- **Run error demo**: shows a progress overlay, then displays an error card with a close button.
